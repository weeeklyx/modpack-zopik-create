# ZopikCreate - сборка модов.
Сборка собрана для выживания на проекте zopik'а с модом Create Aeronautics и открыта для общего пользования.

## Идея и моды
Самый главный мод - Create Aeronautics
После него уже идут много разных аддонов для криейта.

Некоторые из них:
- Steam n' Rails
- Stuff n' Additions
- Create Hypertubes
- Create Connected

и другие

## Как установить
Сборка использует версию Neoforge 1.21.1

### Если у вас лаунчер Modrinth или любой другой, который умеет читать файлы сборок Modrinth:

- Скачайте последнюю версию сборки в формате .mrpack со [страницы релизов](https://github.com/weeeklyx/modpack-zopik-create/releases)
- Откройте свой лаунчер, перетащите файл в него, экспортируйте сборку.
### Более подробная инструкция для Modrinth App
- После загрузки файла откройте меню создания проекта
- Выберите Modpack Base
- Нажмите на Import Modpack и выберите скачанный файл. Сборка сама загрузится в лаунчер

### Если у вас лаунчер, не поддерживающий .mrpack

- Скачайте папку модов в формате .zip со страницы релизов из последней версии модпака
- Установите Neoforge 1.21.1
- Закиньте скачанный архив в папку модов
- Распакуйте архив
